# Setup

**Step 1** - Clone the repository using git clone.

```git clone { url }```

**Step 2** - Install project dependencies-

```pip3 install -r requirements.txt```

**Step 3**- Open the Terminal and type `pythonp3` and init the DB-

```
from app import db
db.create_all()
exit()
```

**Step 4** - Run the app-

```python3 app.py```
